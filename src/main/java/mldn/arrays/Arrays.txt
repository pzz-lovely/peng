

    方法
    static int binarySearch(数据类型[] a,数据类型 key)
    二分查找法，也就是现在对于所有的查找性能的最佳优化。返回数组的索引位置
    static int compare(数据类型[] a,数据类型[] b)

    static 数据类型[] copy(数据类型[] a,数据类型[] b)
    数组拷贝，从原始数组拷贝到新的数组
    static boolean equals()
    数组的长度 要一致
    static int hashCode(数据类型[] a)

自定义查找
    假设有100个长度的数组，那么最多排查100次，如果有100W长度的数组，那么最多排查100W次，那么按照程序的设计概念来讲此时Search()方法时间长度为 "O(n)"
    n为数组的长度。随着内容的增加时间复杂度还会攀升。在整个计算机的世界里，如果按照查找来讲，最快的速度就是 "O(log2N)"
