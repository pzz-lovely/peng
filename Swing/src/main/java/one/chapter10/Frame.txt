    设置标题

    void setIconImage()
    获取或设置iconImage属性，确定图标
    boolean isUndecorated() Undecorated可修饰的

    int getExtendedState() 扩张状态
    void setExtendedState(int state) 以下是状态值
        Frame.NORMAL

        Frame.MAXIMIZED_VERT 最大化
        Frame.MAXIMIZED_BOTH 两边水平的
