    边框布局管理器(border layout manager) 是每个JFrame的内容窗格的默认布局管理器


    java.awt.Container extends Component
    Void SetLayout(LayoutManger m)
    为容器设置布局管理器
    Component add(Component c)
    Component add(Component c,Object constraints)
    将组件添加到容器中，并返回组件的引用
        参数 c  要添加的组件
            constraints 布局管理理解的标识符

    java.awt.FlowLayout 流布局
    FlowLayout()
    FlowLayout(int align)
    FlowLayout(int align,int tgap,int vgap)
    创建一个新的FlowLayout对象
        align LEFT,CENTER,RIGHT
        hgap 以像素为单位的水平间距
        vgap 以像素为单位的垂直间距

    java.awt.ButtonLayout
    BorderLayout()
    BorderLayout(int hgap,int vgap)
    构建一个新的BorderLayout对象

    java.awt.GridLayout 网格
    GridLayout(int rows,int columns)
    GridLayout(int rows,int columns,int hgap,int vgap)
    构建一个新的GridLayout对象
        row 网格的行数
        columns 网格的列数
        hgap 以像素为单位的水平间距
        vgap 以像素为单位的垂直间距
